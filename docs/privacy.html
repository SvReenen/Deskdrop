Privacy Policy

Last updated: April 12, 2026

Overview

Deskdrop is an open-source Android keyboard. Your privacy is the top priority. Deskdrop does not collect, transmit, or sell any personal data. There is no telemetry, no analytics, and no tracking of any kind.

Data that stays on your device

All of the following data is stored locally on your device and never leaves it:

  • Your typing input and learned words
  • Clipboard history
  • Conversation history
  • App settings and preferences
  • Reminders

Network connections

Deskdrop only makes network requests when you explicitly use AI features. These requests go to servers that you configure yourself:

  • Local AI server (Ollama, LM Studio, etc.) - Your prompts are sent to your own server on your local network or via Tailscale. No third party is involved.
  • Cloud AI providers (Gemini, Groq, OpenRouter, Anthropic, OpenAI) - If you choose to configure a cloud provider, your prompts are sent to that provider's API. This is entirely optional and requires you to enter your own API key. Refer to each provider's privacy policy for how they handle data.
  • Whisper server - If you configure voice input with a self-hosted Whisper server, audio recordings are sent to your server for transcription and deleted immediately after.
  • MCP servers - If you configure external tool servers (e.g. Home Assistant), requests are sent to those servers. This is optional and user-configured.

Deskdrop never sends data to any server that you have not explicitly configured.

API key storage

API keys are encrypted using AES-256-GCM via Android's EncryptedSharedPreferences. Keys are never logged and are only held in memory during active use.

Permissions

Deskdrop requests only the permissions necessary for its features:

  • Internet - To connect to your configured AI servers
  • Microphone - Only when you use voice input (Whisper or Google Speech)
  • Contacts - Only when the contacts tool is enabled (opt-in, off by default)
  • Calendar - Only when the calendar tool is enabled (opt-in, off by default)
  • Phone - Only when the phone call tool is enabled (opt-in, off by default)
  • SMS - Only when the SMS tool is enabled (opt-in, off by default)

Device action permissions (contacts, calendar, phone, SMS) are all opt-in and disabled by default. They are never accessed unless you explicitly enable them in settings.

Data backup

Android backup is disabled (allowBackup="false"). App data cannot be exported via ADB or cloud backup services.

Children

Deskdrop is not directed at children under 13. We do not knowingly collect data from children.

Open source

Deskdrop is licensed under GPL-3.0. The full source code is available at github.com/SvReenen/Deskdrop. You can verify all privacy claims by reviewing the code.
